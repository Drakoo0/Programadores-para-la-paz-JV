Pregunta 1: ¿Qué es una ruta en Express?

Respuesta: Una dirección que define cómo responde 
el servidor a una solicitud.


Pregunta 2: ¿Qué método HTTP se usa para consultar?

Respuesta: GET


Pregunta 3: ¿Qué método HTTP se usa para enviar?

Respuesta: POST


Pregunta 4: ¿Qué significa API?

Respuesta: Interfaz que permite que sistemas o 
aplicaciones se comuniquen entre sí.
